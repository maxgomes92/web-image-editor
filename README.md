# Appostrophe Coding Challenge

[Full Stack Technical Assignment](https://appostrophe.notion.site/Full-Stack-Technical-Assignment-1bd1f81fe41280b48137e2cd079f62d4)

This repository contains a simple image editor.

## Features

1. Creating a scrollable and zoomable canvas
    - When the user zooms, using either a scroll event or a pinch gesture on a trackpad, the zoom should be anchored to the position of the mouse pointer on the canvas.
2. Allow the user to select a photo from their computer
3. Add the selected photo to the canvas
4. Enable item selection by clicking on an item, and deselection by clicking outside it
5. Allow items to be moved around the canvas by clicking and dragging
6. Provide handles for scaling and rotating items
